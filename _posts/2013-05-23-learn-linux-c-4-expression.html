    @2：赋值运算符
        C语言中，赋值不是表达式是运算符
        比如 
            int i, j;
            float x, y;
            i = j = x = y = 10;
        最终的结果就是i=10; j = 10; x = 10.0; y = 10.0;
        解释：不建议，会出现默写不理解和错误
        Lvalue required 左值错误
        复合赋值
            int j, i;
            i = i + 3;
            i += 3;
            j = 10;
            i += j %= 9;
    @3：自增自减运算符
        ++ --会有副作用，改变操作数
        前 ++ --和后++ --是不一样的
        前 ++i mean: i + 1;副作用 i = i + 1;
        后 i++ mean:i     ;副作用 i = i + 1;
