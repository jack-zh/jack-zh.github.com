数组

    @1：一维数组
        一维数组初始化：
            int a[10] = {1, 2, 3, 4, 5, 6, 7, 8, 9, 0};
            int a[] = {1, 2, 3, 4, 5, 6, 7, 8, 9, 0};/*省略长度*/
            int a[10] = {1, 2, 3, 4};/*其余的值为0*/
            int a[10] = {0};/*其余的值为0 方便的初始化*/
        变化的数组长度
            for (i = 0; i < sizeof(a) / sizeof(a[0]); i++)
                ;
            or:
                #define SIZE (sizeof(a) / sizeof(a[0]))
    @2：多维数组
        多维数组的初始化：
            int m[2][3] = {
                {1,2,3},
                {2,3,4}
            };
            int m[2][3] = {
                {1,2}
            };/*各种置为0*/
            相当于：
            int m[2][3] = {
                {1,2,0},
                {0,0,0}
            };
        @3：常量数组
            const int months[] = 
            {12,23,34,45,56,67,78,89}
